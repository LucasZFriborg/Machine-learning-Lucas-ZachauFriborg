# Movie Recommendation System

## Overview
This project implements a **movie recommendation system** as part of a Machine Learning assignment. The goal is to recommend five similar movies based on a given input movie.

The system uses a **content-based filtering approach**, where recommendations are generated by comparing movie features such as genres and user-generated tags.

---

## Dataset
This project uses the **MovieLens "ml-latest" dataset**, which includes:
- `movies.csv` – Movie titles and genres  
- `tags.csv` – User-generated tags  
- `ratings.csv` – (not used in this implementation)  
- `links.csv` – (not used)
  
The dataset is **not included in this repository** due to its large size.

### How to get the dataset:
1. Download it from:  
   https://grouplens.org/datasets/movielens/latest/
2. Extract the contents
3. Place the folder like this:
Data/ml-latest/
movies.csv
tags.csv
ratings.csv
links.csv
